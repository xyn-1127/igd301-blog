Lecture HW2 — CAVE System

1. Difference between a CAVE system and Head-Mounted Displays (HMDs) A CAVE (Cave Automatic Virtual Environment) is a room-scale immersive display: stereo projections are shown on multiple surfaces (walls, floor, sometimes ceiling), so the user stands inside the display space. The system is typically combined with position tracking so that the rendered viewpoint updates with the user’s motion and preserves a natural perspective. An HMD delivers the visuals through a head-worn device, with the virtual scene presented individually to each user. The user’s viewpoint is updated primarily through head tracking. ...
